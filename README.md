# calculator-
#A web-based calculator modelled on the Casio fx-911EX, built with HTML, CSS, and JavaScript.  
It performs basic arithmetic operations and demonstrates user interaction and dynamic UI.
## Features
- Addition, subtraction, multiplication, division
- triganomatry (sin , cos , tan , but no inverse triganometry)
- more feature includes Log , in ,root,powers , negatives ,negative powers , 
- Clean button UI
- Works in modern browsers
## How to Use
1. Clone the repository:

2. Open `calculator_model2.html` in a browser.

## Tech Used
- HTML
- CSS
- JavaScript
